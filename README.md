## 📂 Projects

### 1. IoT-AI Health Monitor & Gesture Control System
*ESP32 | Python | PyTorch | Sensor Fusion | Cloud Dashboard*

A dual-function ESP32 system that streams real-time heart rate and body temperature to a cloud dashboard with AI-driven anomaly detection, while also recognizing fist open/close gestures with 95% accuracy for touchless device control. Trained the gesture model, exported as .pth/.h files, and integrated sensors (max30100, ds18b20).

### 2. PolicySphere – AI Policy Impact Analyzer
*Python | Flask | Google Gemini API | JavaScript | Chart.js*

Full-stack web app where users paste policy text → Gemini AI predicts economic impacts (GDP, inflation, sector effects) for 5 stakeholder groups → outputs risk scores, 4-step action plans, and downloadable reports. Optimized prompts to reduce token usage by ~70%, enabling ~4,000 free API calls/day with fallback mock-data architecture.

### 3. AI Interview Malpractice Detection System
*COCO-SSD | Object Detection | Real-time | Flask*

Lightweight object detection model that runs on a candidate's browser, detecting unauthorized objects (phone, book, extra person), tab switching, and gaze deviation in real-time. Deployed at UptoSkills Online, reducing manual proctoring effort by ~80%.

### 4. Face Recognition Attendance System
*OpenCV | Haar Cascade | KNN | Flask | CSV Logging*

Real-time system that detects faces using Haar Cascade and identifies registered students with a KNN classifier, achieving 85% accuracy across varied lighting conditions. Automatically logs timestamped attendance to CSV with voice confirmation, eliminating manual roll-call.

### 5. Machine Learning Model Suite
*Scikit-learn | NumPy | Pandas | KNN | Decision Trees | K-Means | Regression*

Benchmarked KNN, Decision Trees, K-Means, and Regression models on real-world datasets comparing accuracy, precision, and recall. Built end-to-end preprocessing pipelines including null handling, feature scaling, and train/test splits.
